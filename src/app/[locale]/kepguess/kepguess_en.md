### Technical Highlights

KepGuess is built with modern, scalable technologies:

- **Frontend:** Flutter 3.x for beautiful cross-platform mobile experience
- **Backend:** Supabase (PostgreSQL) for robust data management
- **Authentication:** Google OAuth for secure, abuse-resistant access
- **State Management:** Riverpod 2.x for type-safe, testable code
- **Notifications:** Automated email system using Supabase Edge Functions + Resend API
- **Admin Panel:** Next.js-based dashboard for content management

### Design Philosophy

KepGuess follows a **minimal, clean, and prediction-card-focused** design:

- Dark theme with vibrant green accents (#13EC5B)
